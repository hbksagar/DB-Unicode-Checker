# DB-Unicode-Checker

## Requirements:
1. MySQL
2. Python
3. SQLAlchemy

## Configure:
1. Update db_name, db_username, db_password and db_host in config.py

## File Execution: 
```python source.py```

## Required system packages:
1. mysql
2. libmysqlclient-dev
3. python-dev
4. pip

## Run:
pip install -r requirements.txt
